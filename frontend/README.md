# FlowMate - Autonomous Financial AI Agent

Your autonomous financial operating system on Flow blockchain.

## What is FlowMate?

FlowMate is an AI-powered financial agent that handles all your financial operations autonomously:

- **💰 Autonomous Savings** - Automatically save money based on your goals
- **💸 Smart Payments** - Send money to friends, family, and services automatically
- **🔄 Token Management** - Swap and stake assets without manual steps
- **📅 Recurring Operations** - Set up automated transfers, investments, and bills
- **🤖 Natural Language** - Just tell FlowMate what you want in plain English

## Getting Started

### Prerequisites

- Node.js 18+
- npm or bun

### Installation

```bash
npm install
# or
bun install
```

### Development

```bash
npm run dev
# or
bun run dev
```

The app will run at `http://localhost:8080`

### Building

```bash
npm run build
```

### Testing

```bash
npm run test
npm run test:watch
```

## Features

- **Dashboard** - Overview of your wealth, goals, and recent operations
- **AI Agent Chat** - Communicate with FlowMate in natural language
- **Configuration** - Set autonomy levels (Manual, Assisted, Autonomous)
- **Profile Management** - Manage your financial profile and wallet

## Tech Stack

- **Frontend**: React 18 + TypeScript
- **UI Framework**: Tailwind CSS + shadcn/ui
- **State Management**: TanStack Query
- **Routing**: React Router v6
- **Animations**: Framer Motion
- **Build Tool**: Vite
- **Testing**: Vitest + Playwright
